jun. 2022 - Ago. 2022 Blides Publicity

Ambos proyectos involucraron el uso de tecnologías web como HTML, CSS, JavaScript y PHP. Oct. 2022 - Mar. 2023 Genius Digital Solution
Developer Mobile y Web

Tuve la oportunidad de trabajar en dos proyectos emocionantes de programación. El segundo proyecto fue una tienda en línea de venta de cannabis desarrollada con Next.js. En el proyecto de la aplicación móvil, fui parte de el equipo de desarrollo con React Native para crear una aplicación móvil multiplataforma que funcionara en dispositivos iOS y Android. Diseñé una interfaz de usuario intuitiva y fácil de usar que permitía a los empleados ingresar su información de inicio de sesión y registrar su hora de entrada y salida en un sistema centralizado. En el segundo proyecto, fui parte del equipo de trabajado utilizando Next.js para desarrollar una tienda en línea de venta de cannabis. Se diseño una interfaz de usuario atractiva y fácil de usar que permitía a los usuarios buscar y comprar productos en línea.

Mar. 2023 - Actualmente Yufi
Developer Full Stack

En mi último trabajo, fui parte del equipo de desarrollo de un emocionante proyecto como programador full-stack que involucró el uso de Next.js y Laravel para desarrollar una plataforma para solicitudes de financiamiento para los propietarios de Airbnbs. En este proyecto, trabajé en el desarrollo de características de la plataforma, implementé nuevas funcionalidades y mejoré la experiencia del usuario. Utilizamos Laravel para construir el backend de la plataforma, donde implementé una API RESTful para manejar todas las solicitudes y respuestas de los clientes. También trabajé en la integración de Laravel con una base de datos MySQL y desarrollé modelos, controladores y migraciones para administrar la información de los usuarios y las solicitudes de financiamiento. En cuanto al frontend, trabajé con Next.js para construir una interfaz de usuario fácil de usar y atractiva. Diseñé páginas de inicio de sesión, registro, dashboard y otras páginas necesarias para el flujo de la plataforma. También trabajé en la integración de algunas frameworks populares de Css, como Tailwind Css, para mejorar la apariencia del sitio web.
